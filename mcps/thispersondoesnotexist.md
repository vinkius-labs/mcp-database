## Description
Connect your AI agent to the **ThisPersonDoesNotExist** engine and generate unique, AI-synthesized human faces directly through natural conversation.

### What you can do

- **Instant Face Generation** — Fetch a unique, GAN-generated 1024x1024 JPEG image of a person who does not exist.
- **High-Quality Assets** — Receive high-resolution images (StyleGAN2) suitable for UI/UX mockups, character design, or testing environments.
- **Privacy-First Personas** — Use synthetic media to create realistic user profiles without using real people's data or identities.

### How it works

1. Subscribe to this server
2. Enter 'PUBLIC' as your access key (this service is open-access)
3. Start generating faces directly in Claude, Cursor, or any MCP-compatible client

No more searching through stock photo sites for generic faces. Your AI acts as an instant generator for high-fidelity synthetic portraits.

### Who is this for?

- **UI/UX Designers** — quickly populate mockups and prototypes with realistic user avatars.
- **Developers** — generate test data for applications that require profile pictures without privacy concerns.
- **Creative Writers** — visualize characters for stories or role-playing scenarios using AI-generated imagery.

## ❓ FAQ

**Q: Are the people in these images real?**
No. Every image generated by the `get_random_face` tool is synthesized by a StyleGAN2 neural network. These individuals do not exist in the real world.

**Q: What is the resolution of the generated faces?**
The `get_random_face` tool provides high-quality JPEG images at a fixed resolution of 1024x1024 pixels.

**Q: Can I use these images for commercial mockups?**
Yes, these AI-generated faces are commonly used for design prototypes, testing, and creative projects where a human likeness is needed without legal or privacy constraints.

### Claude.ai
Follow the steps below to connect in seconds.

1. Open [claude.ai](https://claude.ai) and sign in to your account.
2. Go to **Customize → Connectors**.
3. Click the **+** button and select "Add custom connector".
4. Paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`) and save.
5. Click the **+** button in any chat and enable **ThisPersonDoesNotExist** under Connectors.

### Cursor
Follow the steps below to connect in seconds.

1. In Cursor, open Settings (`⌘ ,`) → scroll to **Features** → **MCP Servers**.
2. Click **+ Add new MCP Server**.
3. Set Type to "SSE" (or "streamable HTTP"), enter `thispersondoesnotexist` as the name, and paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`).
4. Click **Save** — Cursor will connect and list all **ThisPersonDoesNotExist** tools.

**Configuration:**
```json
{
  "mcpServers": {
    "thispersondoesnotexist": {
      "url": "https://edge.vinkius.com/[TOKEN]/mcp"
    }
  }
}
```
