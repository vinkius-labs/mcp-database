Estimate WebSocket connection limits, server fleet requirements, and network overhead.

## Description
This MCP server provides a specialized engine for planning WebSocket infrastructure. It allows AI agents to calculate the maximum concurrent connections based on memory constraints using `estimate_max_connections`, determine necessary fleet size with `calculate_required_servers`, estimate network egress load via `estimate_broadcast_load`, and quantify heartbeat bandwidth overhead with `calculate_bandwidth_overhead`.


## Available Tools (4)
- **calculate_bandwidth_overhead**: Estimates the bandwidth consumed strictly by heartbeat (ping/pong) traffic
- **calculate_required_servers**: Determines the fleet size needed to support a global target of connections
- **estimate_broadcast_load**: Calculates the network egress load generated by message broadcasting across channels
- **estimate_max_connections**: Calculates the maximum number of concurrent connections a single node can host based on memory constraints

## ❓ FAQ

**Q: How do I estimate the maximum connections for my server?**
Use the `estimate_max_connections` tool by providing your available RAM and the estimated memory footprint per connection.

**Q: Can this help me plan for a global user base?**
Yes, use `calculate_required_servers` to determine how many server instances are needed to support your target total connection count.

**Q: How is network overhead calculated?**
The `calculate_bandwidth_overhead` tool estimates the bandwidth consumed by heartbeat traffic based on your connection count and ping interval.

Follow the steps below to connect in seconds.

3. Click the **+** button and select "Add custom connector".
4. Paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`) and save.
5. Click the **+** button in any chat and enable **WebSocket Capacity Planner** under Connectors.

### Cursor
Follow the steps below to connect in seconds.

1. In Cursor, open Settings (`⌘ ,`) → scroll to **Features** → **MCP Servers**.
2. Click **+ Add new MCP Server**.
3. Set Type to "SSE" (or "streamable HTTP"), enter `websocket-capacity-planner` as the name, and paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`).
4. Click **Save** — Cursor will connect and list all **WebSocket Capacity Planner** tools.
