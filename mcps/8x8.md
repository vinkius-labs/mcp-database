Power your cloud communications with AI-driven call management, voicemail access, and team messaging across every channel.

**Category:** communication-messaging

## Description
Connect your **8x8** Connect account to any AI agent and manage SMS, chat, voice, and verification workflows through natural conversation.

### What you can do

- **SMS Messaging** — Send text messages to any phone number worldwide and track delivery status
- **Chat Messaging** — Send messages through WhatsApp, Viber, and LINE channels
- **Voice Calls** — Initiate text-to-speech voice calls and check call status
- **Phone Verification** — Send OTP codes via SMS and validate user submissions
- **Account Management** — List sub-accounts and check account balance
- **Delivery Tracking** — Monitor SMS and voice call delivery status in real time

### How it works

1. Subscribe to this server
2. Enter your **API Key** and **Sub Account ID** from your 8x8 Connect dashboard
3. Start sending messages from Claude, Cursor, or any MCP client

### Who is this for?

- **Customer Support** — send SMS confirmations and initiate voice callbacks from AI
- **Product Teams** — integrate phone verification into onboarding flows
- **Operations** — monitor account balance and sub-account usage programmatically


## Available Tools
- **check_8x8_status**: Verify API connectivity
- **check_verification**: Verify an OTP code
- **get_account_balance**: Get account balance
- **get_call_status**: Get voice call status
- **get_sms_status**: Get SMS delivery status
- **list_sub_accounts**: List sub-accounts
- **send_chat_message**: Send a chat message
- **send_sms**: Send an SMS message
- **send_verification**: Send a verification code
- **send_voice_message**: Make a text-to-speech call
