## Description
Connect your **Akeneo PIM** instance to your AI agent to unlock professional product data orchestration. From auditing product technical metadata and localized descriptions to managing attribute groups and category trees, your agent handles your e-commerce catalog through natural conversation.

### What you can do

- **Product Orchestration** — List and retrieve details for products using identifiers or UUIDs, and update attributes seamlessly
- **Taxonomy Management** — List and audit category trees to ensure your product classification is always consistent
- **Attribute Oversight** — List available attributes and families to understand your PIM's data structure
- **Channel & Locale Auditing** — Retrieve configured channels and locales to manage multi-market product data
- **Catalog Insights** — Quickly identify incomplete product records or audit attribute values directly from your chat interface

### How it works

## ❓ FAQ

**Q: How do I find my Akeneo API credentials?**
Log in to Akeneo, navigate to **Connect** > **Connection settings**, and create a new connection. You will generate a Client ID, Secret, and a system user with its own Username and Password for the API.

**Q: Can I update product attributes via the agent?**
Yes! Use the `update_product` tool and provide the product identifier along with the attributes you want to change. Your agent will submit a PATCH request to synchronize the data.

**Q: What is the difference between SKU and UUID in Akeneo?**
The SKU (identifier) is your internal product code. The UUID is a unique system identifier generated by Akeneo. While both can be used, UUIDs are recommended for permanent links as identifiers can technically be changed.

### Claude.ai
Follow the steps below to connect in seconds.

1. Open [claude.ai](https://claude.ai) and sign in to your account.
2. Go to **Customize → Connectors**.
3. Click the **+** button and select "Add custom connector".
4. Paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`) and save.
5. Click the **+** button in any chat and enable **Akeneo PIM** under Connectors.

### Cursor
Follow the steps below to connect in seconds.

1. In Cursor, open Settings (`⌘ ,`) → scroll to **Features** → **MCP Servers**.
2. Click **+ Add new MCP Server**.
3. Set Type to "SSE" (or "streamable HTTP"), enter `akeneo-pim` as the name, and paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`).
4. Click **Save** — Cursor will connect and list all **Akeneo PIM** tools.

**Configuration:**
```json
{
  "mcpServers": {
    "akeneo-pim": {
      "url": "https://edge.vinkius.com/[TOKEN]/mcp"
    }
  }
}
```
