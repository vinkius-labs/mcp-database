# oboloo MCP Server

Manage procurement and supplier relationships with contract tracking, spend analysis, and compliance monitoring for buying teams.

[![View on Vinkius](https://img.shields.io/badge/View_on-Vinkius-blue?style=for-the-badge)](https://vinkius.com/mcp/oboloo)

## Overview
**Category:** erp-operations
**Tools Count:** 12

## Description
Connect your **Oboloo** account to any AI agent and manage procurement operations.

### What you can do

- **Supplier Management** — List suppliers and view profiles with risk scores
- **Contract Tracking** — Browse and inspect procurement contracts
- **Savings Analysis** — View recorded cost savings
- **Project Management** — Track procurement projects and timelines
- **Approvals** — View pending procurement approvals
- **Risk Assessment** — List supplier and contract risks
- **Spend Analysis** — Get category and supplier spend breakdowns


## Available Tools
- **check_oboloo_status**: Verify API connectivity
- **get_contract**: Get contract details
- **get_project**: Get project details
- **get_spend_analysis**: Get spend analysis
- **get_supplier**: Get supplier details
- **list_approvals**: List pending approvals
- **list_categories**: List spend categories
- **list_contracts**: List all contracts
- **list_projects**: List procurement projects
- **list_risks**: List supplier risks
- **list_savings**: List savings
- **list_suppliers**: List all suppliers
