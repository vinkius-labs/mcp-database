## Description
Connect your **Besitos Corp** partner account to any AI agent and orchestrate your mobile gaming and user engagement workflows through natural conversation.

### What you can do

- **Game Discovery** — List and inspect available mobile games in the Besitos marketplace for integration.
- **Offer Management** — Query reward offers and milestones to optimize user incentives.
- **User Activity Tracking** — Retrieve detailed gaming activity and history for specific users.
- **Reward Oversight** — Monitor user reward redemptions and historical earnings.
- **Campaign Analytics** — Access real-time reports on user engagement and monetization performance.
- **Fraud Prevention Monitoring** — Verify the status of campaigns and user interactions through the integrated fraud layer.

### How it works

1. Subscribe to this server
2. Enter your Besitos Partner API Key
3. Start managing your gaming and rewards ecosystem from Claude, Cursor, or any MCP-compatible client

### Who is this for?

- **Fintech & Loyalty Platforms** — quickly check game availability and reward structures for integration.
- **Marketing Leads** — retrieve engagement reports and campaign metadata straight from their workspace.
- **Account Managers** — verify user activity and reward history for support and optimization.


## Available Tools (10)
- **get_campaign**: Get details of a specific campaign
- **get_engagement_report**: Retrieve a summary of user engagement metrics
- **get_game**: Get details of a specific game
- **get_offer**: Get details of a specific reward offer
- **get_revenue_report**: Retrieve a summary of monetization and revenue data
- **list_campaigns**: List all reward-based engagement campaigns
- **list_games**: List all available mobile games in the marketplace
- **list_offers**: List all reward offers and milestones
- **list_user_activity**: Retrieve gaming activity for a specific user
- **list_user_rewards**: Retrieve reward history for a specific user

## ❓ FAQ

**Q: Can I check which games are currently trending in the marketplace?**
Yes! Use the `list_games` tool to see all available titles. You can then use `get_game` to see specific metadata and performance metrics for each game.

**Q: How do I monitor the revenue generated by my gaming channel?**
Simply ask the agent to `get_revenue_report`. It will retrieve a summary of monetization data, including total earnings and performance by campaign.

**Q: Does the integration allow me to verify a specific user's reward status?**
Yes. Use the `list_user_rewards` tool with the User ID. Your agent will fetch the full history of earned rewards and pending redemptions for that user.


## Installation & Usage

This MCP server is fully hosted and managed by **[Vinkius Cloud](https://vinkius.com)**, providing a zero-setup, high-performance, and secure execution environment. You do not need to manage local servers or dependencies. Simply connect your AI agent to the Vinkius Edge network using the instructions below.

2. Connect to the Vinkius Cloud to start using it: [cloud.vinkius.com/connect](https://cloud.vinkius.com/connect)

### Claude.ai
Follow the steps below to connect in seconds.

1. Open [claude.ai](https://claude.ai) and sign in to your account.
2. Go to **Customize → Connectors**.
3. Click the **+** button and select "Add custom connector".
4. Paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`) and save.
5. Click the **+** button in any chat and enable **Besitos Corp** under Connectors.

### Cursor
Follow the steps below to connect in seconds.

1. In Cursor, open Settings (`⌘ ,`) → scroll to **Features** → **MCP Servers**.
2. Click **+ Add new MCP Server**.
3. Set Type to "SSE" (or "streamable HTTP"), enter `besitos-corp` as the name, and paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`).
4. Click **Save** — Cursor will connect and list all **Besitos Corp** tools.

**Configuration:**
```json
{
  "mcpServers": {
    "besitos-corp": {
      "url": "https://edge.vinkius.com/[TOKEN]/mcp"
    }
  }
}
```
