## Description
Restpack provides powerful tools for web-to-document conversion. This MCP server allows your AI to capture live web content and transform it into structured formats through natural conversation.

### What you can do

- **HTML to PDF** — Convert live URLs or raw HTML strings into high-quality PDF documents with full CSS support and custom page sizes.
- **Web Screenshots** — Capture full-page, viewport, or specific element screenshots in PNG, JPG, or PDF formats with retina support.
- **Advanced Layout Control** — Manage margins, orientation (portrait/landscape), and inject custom CSS or JS before rendering.
- **Document Metadata** — Add authors, encryption, and passwords to your generated PDFs for professional use.
- **Media Emulation** — Switch between 'screen' and 'print' media types to ensure the document looks exactly as intended.

### How it works

1. Subscribe to this server
2. Enter your Restpack API Token
3. Start generating documents and screenshots from Claude, Cursor, or any MCP-compatible client

### Who is this for?

- **Developers & DevOps** — Automate UI testing screenshots or generate technical documentation directly from the code editor.
- **Researchers & Analysts** — Archive web pages as PDFs for offline reference and data preservation without leaving the chat.
- **Marketing Teams** — Generate high-quality social media previews or website mockups instantly using simple prompts.


## Available Tools
- **capture_screenshot**: Returns a CDN URL.

Capture a screenshot of a web page
- **convert_html_to_pdf**: Returns a CDN URL.

Convert a web page or HTML to PDF
- **create_browser_mockup**: Returns a CDN URL.

Convert an image into a browser mockup
- **get_usage**: Get API usage statistics
