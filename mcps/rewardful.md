## Description
Connect your **Rewardful** account to any AI agent to automate your affiliate marketing and referral workflows through natural conversation.

### What you can do

- **Campaign Management** — List all active campaigns, create new ones, and update reward structures or URLs.
- **Affiliate Operations** — Register new affiliates, update their profiles, and generate SSO Magic Links for instant dashboard access.
- **Commission Tracking** — Query specific commissions, update statuses, or delete records to keep your ledger accurate.
- **Payouts & Referrals** — List pending payouts, process payments, and monitor referral traffic generated by your partners.
- **Links & Coupons** — Manage affiliate links and coupons to optimize conversion tracking for your marketing campaigns.

### How it works

1. Subscribe to this server
2. Enter your Rewardful Secret API Key
3. Start managing your partner ecosystem from Claude, Cursor, or any MCP-compatible client

### Who is this for?

- **Marketing Managers** — quickly check campaign performance and list active affiliates without leaving the chat.
- **Growth Leads** — automate the creation of new affiliate links and coupons for influencers and partners.
- **Finance Teams** — audit commissions and process payouts efficiently using simple natural language commands.

## ❓ FAQ

**Q: How can I quickly see all my active affiliate campaigns?**
You can use the `list_campaigns` tool. It will retrieve all campaigns configured in your Rewardful account, including their reward types and settings.

**Q: Is it possible to pay out a pending commission through the AI?**
Yes, you can use the `pay_payout` tool by providing the specific Payout ID. This will mark the payout as paid in your Rewardful dashboard.

**Q: Can I generate a direct login link for an affiliate?**
Absolutely. Use the `generate_affiliate_sso` tool with the Affiliate ID. It generates a one-time Magic Link that expires in 1 minute for secure access.

### Claude.ai
Follow the steps below to connect in seconds.

1. Open [claude.ai](https://claude.ai) and sign in to your account.
2. Go to **Customize → Connectors**.
3. Click the **+** button and select "Add custom connector".
4. Paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`) and save.
5. Click the **+** button in any chat and enable **Rewardful** under Connectors.

### Cursor
Follow the steps below to connect in seconds.

1. In Cursor, open Settings (`⌘ ,`) → scroll to **Features** → **MCP Servers**.
2. Click **+ Add new MCP Server**.
3. Set Type to "SSE" (or "streamable HTTP"), enter `rewardful` as the name, and paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`).
4. Click **Save** — Cursor will connect and list all **Rewardful** tools.

**Configuration:**
```json
{
  "mcpServers": {
    "rewardful": {
      "url": "https://edge.vinkius.com/[TOKEN]/mcp"
    }
  }
}
```
