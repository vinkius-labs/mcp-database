## Overview

**Category:** [marketing-automation](../categories/marketing-automation.md)

Manage affiliate and referral programs via Rewardful — list campaigns, register affiliates, and track commissions directly from any AI agent.

## Description
Connect your **Rewardful** account to any AI agent to automate your affiliate marketing and referral workflows through natural conversation.

### What you can do

- **Campaign Management** — List all active campaigns, create new ones, and update reward structures or URLs.
- **Affiliate Operations** — Register new affiliates, update their profiles, and generate SSO Magic Links for instant dashboard access.
- **Commission Tracking** — Query specific commissions, update statuses, or delete records to keep your ledger accurate.
- **Payouts & Referrals** — List pending payouts, process payments, and monitor referral traffic generated by your partners.
- **Links & Coupons** — Manage affiliate links and coupons to optimize conversion tracking for your marketing campaigns.

### How it works

1. Subscribe to this server
2. Enter your Rewardful Secret API Key
3. Start managing your partner ecosystem from Claude, Cursor, or any MCP-compatible client

### Who is this for?

- **Marketing Managers** — quickly check campaign performance and list active affiliates without leaving the chat.
- **Growth Leads** — automate the creation of new affiliate links and coupons for influencers and partners.
- **Finance Teams** — audit commissions and process payouts efficiently using simple natural language commands.


## Available Tools
- **create_affiliate_coupon**: Create a new affiliate coupon
- **create_affiliate_link**: Create a new affiliate link
- **create_affiliate**: Create a new Rewardful affiliate
- **create_campaign**: Create a new Rewardful campaign
- **delete_commission**: Delete a commission
- **generate_affiliate_sso**: Generate an SSO Magic Link for an affiliate
- **get_affiliate_coupon**: Retrieve a specific affiliate coupon
- **get_affiliate_link**: Retrieve a specific affiliate link
- **get_affiliate**: Retrieve a specific Rewardful affiliate
- **get_campaign**: Retrieve a specific Rewardful campaign
- **get_commission**: Retrieve a specific commission
- **get_payout**: Retrieve a specific payout
- **list_affiliate_coupons**: List all affiliate coupons
- **list_affiliate_links**: List all affiliate links
- **list_affiliates**: List Rewardful affiliates
- **list_campaigns**: List all Rewardful campaigns
- **list_commissions**: List all commissions
- **list_payouts**: List all payouts
- **list_referrals**: List Rewardful referrals
- **pay_payout**: Mark a payout as paid
- **update_affiliate_link**: Update an affiliate link token
- **update_affiliate**: Update an existing Rewardful affiliate
- **update_campaign**: Update an existing Rewardful campaign
- **update_commission**: Update a commission
