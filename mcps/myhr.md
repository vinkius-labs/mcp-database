# MyHR MCP Server

Simplify employee management in New Zealand and Australia with leave tracking, timesheets, and HR compliance tools.

[![View on Vinkius](https://img.shields.io/badge/View_on-Vinkius-blue?style=for-the-badge)](https://vinkius.com/mcp/myhr)

## Overview
**Category:** productivity
**Tools Count:** 10

## Description
Connect your **MyHR** account to any AI agent and manage human resources through natural conversation.

### What you can do

- **Employees** — List, create, and view detailed employee profiles
- **Absences** — Track leave requests and create new absence records
- **Timesheets** — Browse employee timesheet entries
- **Configuration** — List departments, offices, marital statuses, and gender options

### Who is this for?

- **HR Teams** — automate employee onboarding
- **Managers** — track team absences and timesheets


## Available Tools
- **create_absence_request**: Create a new absence request
- **create_employee**: Create a new employee
- **get_employee_details**: Get employee details
- **list_absences**: List all employee absences
- **list_departments**: List organization departments
- **list_employees**: List all employees
- **list_gender_options**: List gender options
- **list_marital_statuses**: List marital status options
- **list_offices**: List organization offices
- **list_timesheets**: List all timesheets
