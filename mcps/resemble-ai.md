## Description
Connect your **Resemble AI** account to any AI agent to generate, manage, and protect high-fidelity synthetic speech through natural conversation.

### What you can do

- **Voice Synthesis** — Create high-quality audio clips from text using custom or system voices with full SSML support.
- **Speech-to-Speech** — Transform source audio into a target voice while preserving the original emotion, intonation, and timing.
- **Project Organization** — List, create, and manage projects to keep your audio assets and clips organized.
- **Voice Management** — List available voices, create new custom voice profiles, and manage training recordings.
- **AI Safety & Security** — Detect deepfakes in audio files and apply or verify digital watermarks to ensure content authenticity.

### How it works

1. Subscribe to this server
2. Enter your Resemble AI API Token
3. Start generating and managing AI voices from Claude, Cursor, or any MCP-compatible client

### Who is this for?

- **Content Creators** — Generate voiceovers and localized content instantly without leaving your workflow.
- **Developers** — Integrate advanced voice synthesis and deepfake detection directly into your development environment.
- **Security Teams** — Verify the authenticity of audio files and detect synthetic manipulations using built-in safety tools.

## ❓ FAQ

**Q: Can I use SSML tags when creating an audio clip?**
Yes! The `create_clip` tool fully supports SSML in the body field, allowing you to control pitch, rate, and emphasis for more natural-sounding speech.

**Q: How can I transform an existing audio file into a different voice?**
Use the `speech_to_speech` tool. Provide the target `voice_uuid` and a Base64 encoded audio sample or URL, and the AI will re-voice the content while keeping the original performance characteristics.

**Q: Is there a way to check if an audio file is a deepfake?**
Absolutely. The `detect_deepfake` tool allows you to submit an audio URL to Resemble's security engine to analyze whether the speech was generated by AI.


## Installation & Usage

This MCP server is fully hosted and managed by **[Vinkius Cloud](https://vinkius.com)**, providing a zero-setup, high-performance, and secure execution environment. You do not need to manage local servers or dependencies. Simply connect your AI agent to the Vinkius Edge network using the instructions below.

1. View installation instructions and explore the server: [https://vinkius.com/mcp/resemble-ai](https://vinkius.com/mcp/resemble-ai)
2. Connect to the Vinkius Cloud to start using it: [cloud.vinkius.com/connect](https://cloud.vinkius.com/connect)

### Claude.ai
Follow the steps below to connect in seconds.

1. Open [claude.ai](https://claude.ai) and sign in to your account.
2. Go to **Customize → Connectors**.
3. Click the **+** button and select "Add custom connector".
4. Paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`) and save.
5. Click the **+** button in any chat and enable **Resemble AI** under Connectors.

### Cursor
Follow the steps below to connect in seconds.

1. In Cursor, open Settings (`⌘ ,`) → scroll to **Features** → **MCP Servers**.
2. Click **+ Add new MCP Server**.
3. Set Type to "SSE", enter `resemble-ai` as the name, and paste the MCP server link (`https://edge.vinkius.com/[TOKEN]/mcp`).
4. Click **Save** — Cursor will connect and list all **Resemble AI** tools.

**Configuration:**
```json
{
  "mcpServers": {
    "resemble-ai": {
      "url": "https://edge.vinkius.com/[TOKEN]/mcp"
    }
  }
}
```
